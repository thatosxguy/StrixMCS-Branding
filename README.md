# StrixMCS Branding

**Order of the Strix — Edition II**

> *Scientia · Societas · Ascensus*

The canonical brand system for StrixMCS. This repo holds the brand guidelines
document and the official mark assets. The guidelines are the single source of
truth — when something here disagrees with a deck, doc, or dashboard, the
guidelines win.

## View the guide

- **Locally:** open [`index.html`](index.html) in any browser. It is fully
  self-contained — the seal is embedded and the only external dependency is the
  IBM Plex web font (loaded from Google Fonts), so it renders offline too apart
  from the typeface.
- **Live (if GitHub Pages is enabled):**
  `https://thatosxguy.github.io/StrixMCS-Branding/`

## The system at a glance

The guide is the authority; this is only a map of it.

- **The Order** — what the mark stands for and how it's read.
- **One mark** — the seal, its clear space, and how it must be used.
- **Color — three colors, held apart** — disciplined, not monochrome; each
  color has exactly one job:

  | Color | Hex | Role |
  |-------|-----|------|
  | Black | `#000000` | The field. Absolute, everywhere. |
  | Near-black | `#0B0B0B` | Panels, code, raised surfaces. |
  | White | `#F7F7F7` | All text + display. Zero warmth. |
  | Oxblood | `#6E0808` | Structural accent — borders, rules. Never body text. |
  | Ceremonial Gold | `#BCA687` | Ceremonial accents only — the motto, formal flourishes. |

- **Type — mono leads** — IBM Plex Mono for structure, IBM Plex Sans for prose.
- **The Strix names** — naming conventions and the byline rule.
- **How it reads** — one hand, everywhere: document & config headers, screens &
  dashboards.
